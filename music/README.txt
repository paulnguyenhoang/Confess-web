Đặt file nhạc của bạn vào đây với tên: music.mp3

File nên là:
- Format: MP3, OGG, hoặc M4A
- Dung lượng: < 5MB (nhẹ nhàng, không quá nặng)
- Âm lượng: đã normalize, không quá to
- Nội dung: nhạc nền nhẹ nhàng, lãng mạn

Ví dụ tên file hợp lệ:
- music.mp3 ✓
- background.mp3 (cần đổi tên thành music.mp3)

Sau khi đặt file, chạy lại: npm run dev
